# wahly-a-ojo

A recipe collection with tools to render recipes as a website or PDF.

## What's here

- **Recipes** — stored in a structured format (Markdown or similar)
- **Website renderer** — generates a browsable static site from the recipe collection
- **PDF renderer** — exports individual recipes or the full collection as printable PDFs

## Project name

"A ojo" is a Spanish cooking expression meaning to cook by sight — no strict measurements, no timers. The cook uses their eyes and judgment to decide how much seasoning to add or how long something needs on the heat. Recipes here are guides, not rules.
